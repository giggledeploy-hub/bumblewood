# 🍄 Bumblewood Adventure

An interactive storybook for kids aged 6–9, featuring Fizzwick Bumblewand and her friends!

## Run locally

\`\`\`bash
npm install
npm run dev
\`\`\`

## Build for production

\`\`\`bash
npm run build
\`\`\`

Built files appear in `dist/` — ready to deploy anywhere.
